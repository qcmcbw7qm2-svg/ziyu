# 网络暴力/被网暴

## 场景描述

个人（或品牌主理人）在社交平台遭遇大规模网络攻击，包括但不限于：人肉搜索、恶意剪辑造谣、评论区攻击、私信辱骂、线下骚扰。常见触发场景：言论被断章取义、卷入公共事件、被大V挂人、竞品雇佣水军攻击。

## 严重程度

| 级别 | 触发条件 |
|---|---|
| 🔵 苗头 | 几条负面评论，无组织性，未扩散 |
| 🟡 发酵 | 出现挂人帖/视频，开始有陌生人涌入攻击 |
| 🟠 爆发 | 被大V转发，形成话题，跨平台扩散 |
| 🔴 危机 | 人肉信息泄露、线下骚扰、家人被波及、或出现自残/自杀威胁 |

## 应对步骤

### 1. 冷静期（不做任何公开回应）

**这是最关键的一步。网暴的情绪强度远高于其他危机——受害方处于极度愤怒/恐惧/委屈中，此时任何公开回应都是给对方送弹药。**

- 🔵🟡：至少 4-6 小时
- 🟠🔴：立即启用支持系统（亲友/律师/心理支持），但公开回应至少等 12 小时

这期间只做三件事：
- [ ] 截图保存所有攻击内容（证据）
- [ ] 关掉评论区私信通知（不看不回）
- [ ] 找一个不在舆论场里的朋友说话

### 2. 信息收集（让信任的人做，不要自己刷）

- [ ] 攻击来源：哪个账号/哪个帖子发起的？是否有组织性？
- [ ] 攻击规模：多少条？什么平台？增长速度？
- [ ] 攻击话术：用了什么标签/叙事框架（"渣男""骗子""造假"等）
- [ ] 是否涉及人肉：个人信息是否已泄露？家人是否被波及？
- [ ] 是否有现实威胁：线下骚扰？死亡威胁？报警阈值是否已到？

### 3. 定性判断

| 情况 | 策略 |
|---|---|
| 自己说错了话/做错了事 | 认错 + 道歉 + 不辩解 |
| 被断章取义/恶意剪辑 | 放完整证据 + 简洁澄清 + 不恋战 |
| 纯造谣/污蔑 | 律师函 + 平台投诉 + 不公开吵架 |
| 组织性水军攻击 | 收集证据 + 平台举报 + 法律途径 |
| 已涉及人肉/人身威胁 | 报警 + 律师 + 暂时停更 |

**核心原则**：网暴的本质是把一个人变成「公共事件」。你的目标不是说服施暴者（不可能），而是让围观路人看到真相后离开。

### 4. 回应策略

**不要做的事：**
- ❌ 逐条回复反驳（你一条他十条，永远打不完）
- ❌ 情绪化对骂（给围观者递刀）
- ❌ 卖惨（会被截图二次传播嘲讽）
- ❌ 拉黑每一个喷子（浪费时间且会激化）
- ❌ 让家人朋友帮你下场吵架（扩大战场）

**要做的事：**
- ✅ 如果自己确实有错：发一条诚恳道歉，关评论 24-48h
- ✅ 如果被造谣：发一条带证据的澄清，只说事实不动情绪
- ✅ 如果组织性攻击：走法律程序，不公开喊话
- ✅ 如果已威胁人身安全：报警，不在网上讨论此事

### 5. 执行清单

**证据保全**
- [ ] 截图/录屏所有攻击内容（含时间戳、发布者、平台）
- [ ] 保存人肉信息泄露的截图（如适用）
- [ ] 记录攻击时间线（谁先发的、谁转发的、什么时间爆发）

**平台操作**
- [ ] 开启评论过滤/仅关注可评论
- [ ] 对人身攻击/造谣内容逐一举报
- [ ] 对组织性攻击申请平台介入

**法律准备**
- [ ] 联系律师评估是否达到立案标准
- [ ] 准备律师函（如需要）
- [ ] 如果人肉信息已泄露 → 报警 + 要求平台删除

**个人保护**
- [ ] 暂时关闭私信
- [ ] 社交账号改为仅好友可见或停更几天
- [ ] 如需可暂时搬离住处（如果地址已泄露）
- [ ] 通知家人不看不回不转发

### 6. 恢复节奏

```
第 1-3 天：停更或仅发日常，关评论
第 3-7 天：正常更新，评论选择性开放
第 7 天以后：评估是否完全恢复正常
```

## 特殊场景：公众人物/内容创作者的持续网暴

如果你是靠内容吃饭的（自媒体/主播/博主），网暴不仅是心理打击，还直接威胁收入来源。

额外注意事项：
- **广告/合作方沟通**：主动告知合作方当前情况，提供事实说明，防止对方单方面解约
- **内容策略调整**：停更期间可以准备「回归内容」，等风波过去后发一条高质量内容把注意力拉回来
- **不要用网暴事件做内容**：不要做「我被网暴了」的视频——会引来第二轮攻击和「炒作」指控
- **粉丝管理**：引导粉丝不要攻击对方——粉丝替你骂人最后锅还是你背
