# 员工/内部人出事

## 场景描述
员工或前员工的行为引发舆论危机，包括：员工被曝不当行为、内部聊天记录泄露、离职员工爆料、管理层作风问题。

## 严重程度

| 级别 | 触发条件 |
|---|---|
| 🔵 苗头 | 内部讨论/小范围传播 |
| 🟡 发酵 | 聊天截图被公开，行业群讨论 |
| 🟠 爆发 | 大V/媒体跟进，登上平台热搜 |
| 🔴 危机 | 涉及违法、性骚扰、职场霸凌等红线问题 |

## 应对步骤

### 1. 冷静期
- 不要第一时间解雇当事员工或发切割声明 — 先搞清楚事实
- 所有内部人员不得私下接受采访或公开发表看法

### 2. 信息收集
- [ ] 核实事件的真实性（证据、时间线、涉事人员）
- [ ] 了解传播源头（谁泄露的？为什么泄露？）
- [ ] 评估法律风险（是否涉及违法？公司是否有连带责任？）
- [ ] 了解当事员工的说法

### 3. 定性判断

| 情况 | 策略 |
|---|---|
| 员工个人行为，与公司无关 | 切割 + 说明公司立场和制度 |
| 公司管理问题导致的 | 承认管理责任 + 整改 |
| 前员工恶意爆料/失实 | 澄清事实 + 法律手段 |
| 内部矛盾被公开化 | 内部调解 + 对外统一口径 |

### 4. 回应策略

**关键判断：这件事反映的是个人问题还是公司文化问题？**

- 个人行为：明确切割，同时说明公司制度和价值观
- 制度问题：承认不足，公布整改措施
- 涉及违法：配合调查，不包庇

### 5. 执行清单
- [ ] 律师介入评估法律风险
- [ ] 确定对当事员工的处置方案
- [ ] 写内外部两份声明（内部告知员工，外部告知公众）
- [ ] 统一对外口径，指定唯一发言人
- [ ] 如有需要，主动向监管部门/警方报告
- [ ] 后续公布制度整改结果

## 特别提醒

- 不要护短。包庇一个有问题的员工，毁的是整个品牌
- 不要急着切割。"已离职/已开除"如果没有后续整改措施，等于承认管理失职
- 内部口径和外部口径要一致。对外说的话，内部员工也要能接受
- 女性员工/性别相关事件尤其谨慎，公众对此高度敏感
