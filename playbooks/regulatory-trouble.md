# 监管/合规问题

## 场景描述
监管部门介入调查、收到行政处罚、被媒体曝光合规问题。如：税务问题、环保处罚、消防隐患、营业执照问题、行业资质缺失。

## 严重程度

| 级别 | 触发条件 |
|---|---|
| 🔵 苗头 | 收到监管部门问询/自查通知 |
| 🟡 发酵 | 行政处罚信息公开，被行业媒体关注 |
| 🟠 爆发 | 被主流媒体报道，或被要求停业整顿 |
| 🔴 危机 | 涉及刑事调查、被吊销证照、大规模查封 |

## 应对步骤

### 1. 冷静期
**这类事件没有冷静期。涉及监管部门时，每句话都要经过律师。**
所有的第一反应不是对外发声，而是找律师。

### 2. 信息收集
- [ ] 确认监管部门的身份和调查范围
- [ ] 了解具体涉及什么法规、可能面临什么处罚
- [ ] 律师评估法律风险和应对方案
- [ ] 了解行业内同类事件的处理方式和结果
- [ ] 内部自查：问题是否真实存在？严重程度？

### 3. 定性判断

| 情况 | 策略 |
|---|---|
| 确实存在违规 | 承认 + 配合整改 + 不要狡辩 |
| 标准模糊/有争议 | 积极配合 + 说明行业现状 + 寻求重新认定 |
| 被误判/处罚过重 | 行政复议/诉讼 + 对外低调处理 |
| 恶意举报 | 配合调查证明清白 + 不炒作 |

### 4. 回应策略

**核心原则：涉及监管，低调、配合、不狡辩、不炒作。**

- 不要发"我们被冤枉了"类煽情声明
- 不要在社交媒体上试图制造舆论压力对抗监管
- 公开声明必须经律师审核
- 对外口径：配合调查 + 积极整改 + 承担应负责任

### 5. 执行清单
- [ ] 律师全程参与
- [ ] 指定唯一对外发言人（建议是律师或法务）
- [ ] 内部统一口径，员工不得对外讨论
- [ ] 评估是否需要主动公告（上市公司必须、一般公司视情况）
- [ ] 配合调查，提供完整材料
- [ ] 制定整改方案和时间表
- [ ] 完成整改后主动汇报/公示结果
- [ ] 监测舆论但不主动炒作

## 特别提醒

- 涉及监管事件，社交媒体不是主战场，法律程序才是
- "我们已经请了最好的律师" — 这种话不要说，显得心虚
- 不要在朋友圈/微信群里发泄情绪，任何话都可能被截图
- 同行出事时不要幸灾乐祸，下一个可能就是你
