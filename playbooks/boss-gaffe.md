# 老板/管理层言论翻车

## 场景描述

老板或高管在公开场合、社交媒体、直播、内部会议中的言论引发争议。如：不当言论、价值观争议、「何不食肉糜」式发言、歧视性言论、被断章取义、陈年旧帖被挖坟。

**和 celebrity-gaffe 的区别**：celebrity-gaffe 针对的是内容创作者/主播/网红，核心资产是粉丝和个人品牌。boss-gaffe 针对的是企业老板/高管，核心资产是公司和品牌。老板翻车，公司跟着遭殃——股价、合作方、员工士气都会受到影响。

## 严重程度

| 级别 | 触发条件 | 典型表现 |
|------|----------|----------|
| 🔵 苗头 | 言论被截图讨论，限于行业小圈子 | 同行群/脉脉/即刻讨论 |
| 🟡 发酵 | 跨圈传播，普通网友开始讨论 | 小红书/抖音出现解读帖 |
| 🟠 爆发 | 登上热搜，主流媒体/大V跟进评论 | 微博热搜+评论员文章 |
| 🔴 危机 | 引发全民声讨、品牌抵制、合作方切割 | 股价下跌/经销商退网/员工公开信 |

## 应对步骤

### 1. 冷静期
**不要让当事人继续说话。** 这是最重要的一条。老板翻车之后最常见的二次翻车就是「我解释一下」。

- 关掉当事人的社交媒体权限，至少 24 小时
- 不要让公关发「这是个人观点不代表公司立场」——老板的言论就是公司的言论，切割等于自欺欺人
- 不要让其他高管出来「帮老板说话」——多一个人说话多一个靶子
- 🔵🟡：至少 2 小时，先搞清楚老板到底说了什么、上下文是什么
- 🟠🔴：至少 6 小时，启用危机小组

### 2. 信息收集
- [ ] 回溯言论的完整上下文。有没有断章取义？有没有被剪辑？
- [ ] 老板说这句话的时候，环境是什么？跟谁说的？什么场景？
- [ ] 统计各平台的讨论热度和主流观点
- [ ] 评估对业务的实际影响：合作方有没有来电？销售数据有没有波动？员工什么反应？
- [ ] 了解当事人真实意图：说错了？被误解了？还是真的这么想？
- [ ] 这个言论是否触碰法律红线？（地域歧视、性别歧视、虚假宣传等可能涉及违法）

### 3. 定性判断

| 情况 | 策略 | 案例参考 |
|------|------|----------|
| 被断章取义/恶意剪辑 | 放完整视频/上下文 + 标注被剪掉的部分 | 放原视频，让围观者自己看 |
| 确实说错/口误 | 本人道歉 + 说明本意 + 不辩解 | 直播口嗨「不炒股就赚不到钱」 |
| 暴露真实价值观问题 | 真诚道歉 + 个人反思 + 用行动证明 | 不举例，懂的都懂 |
| 陈年旧帖被翻出 | 承认过去的局限 + 说明现在的改变 | 「那时候我不成熟，现在不会了」 |
| 被恶意针对/挖坟 | 不是所有陈年旧事都值得回应。评估：这次挖坟是冲着老板来的还是冲着公司来的？ | 冲着老板来 → 回应；纯网络考古 → 不理 |

### 4. 回应策略

**最核心的判断：本人道歉还是官方声明？**

| 情况 | 谁出声 | 渠道 | 格式 |
|------|--------|------|------|
| 被断章取义 | 本人 | 同一平台 | 放完整视频，不解释 |
| 口误/说错 | 本人 | 视频或手写信 | 对着镜头说，别念稿 |
| 价值观翻车 | 本人 | 视频+文字声明 | 不要说「但是」 |
| 涉及法律红线 | 官方声明+律师 | 官方渠道 | 严肃、简短 |

**道歉的正确姿势**：
1. 先说「我说了 XXX 这句话」
2. 再说「这句话是错的，因为 XXX」
3. 然后说「我为这句话道歉」
4. 最后说「接下来我会 XXX」
5. 全程不说「但是」「只是」「其实」

**道歉的错误姿势**：
- ❌ 「如果我的言论冒犯了任何人」→ 条件式道歉 = 没道歉
- ❌ 「这句话是 XX 年前说的」→ 等于承认说过，但不觉得错了
- ❌ 「当时只是随口一说/开玩笑」→ 不是借口
- ❌ 「我只是在重复别人的话」→ 最没担当的甩锅

**语气铁律**：老板道歉要像真人说话，不像公关稿。公关可以帮忙改语法错误，但不能把「我错了」改成「本公司深刻反思相关言论可能给部分公众带来的困扰」。前者读者信，后者读者骂。

### 5. 执行清单

**和当事人沟通**
- [ ] 先确认：他/她真的认识到问题了吗？还是不服？
- [ ] 如果当事人不认错 → 危机升级。准备好最坏情况预案
- [ ] 如果当事人认错 → 迅速起草道歉内容，让他/她自己改一遍（要本人经手，不能代笔）

**法律评估**
- [ ] 言论是否触碰法律红线（歧视、诽谤、虚假宣传等）
- [ ] 如果涉法 → 律师必须先看过声明再发

**对外**
- [ ] 撰写道歉/澄清文案，本人过手修改
- [ ] 当事人录制视频或发文
- [ ] 选择发布平台：哪里翻车哪里回应。微博翻车不要在公众号道歉
- [ ] 后续观察舆论变化，评估是否需要二次回应

**对内**
- [ ] 向管理层通报情况，统一认识
- [ ] 向全员发送内部通告，说明事件情况和公司态度
- [ ] 让员工知道公司立场，避免员工在社交媒体上「替公司辩解」

## 回应节奏

```
事发 2-6 小时：确认事实 + 了解当事人态度
事发 6-24 小时：本人道歉/澄清发出
第 2-3 天：观察舆论，决定是否追加回应
第 7 天：用行动（不是言论）重建信任
```

## 特别提醒

- 老板翻车是最难处理的危机之一。老板 = 品牌人格化，你无法切割老板和品牌
- 语气比内容更重要。道歉要像真人说话，不像公告
- 一次翻车可以被原谅，反复翻车就是人设崩塌
- 老板翻车后最容易犯的错误不是不道歉，是道歉之后立刻让老板恢复「正常营业」——翻车后至少低调 1-2 周，不要马上发「日常工作」内容
- 丁磊的案例：道歉信里亲自承认「我们的个别员工做事凭惯性，看到负面消息第一时间就想着去处理掉消息」，这种诚恳比任何公关话术都管用
