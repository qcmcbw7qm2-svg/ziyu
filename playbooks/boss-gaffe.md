# 老板/管理层言论翻车

## 场景描述
老板或高管在公开场合、社交媒体、内部会议的言论引发争议。如：不当言论、价值观争议、"何不食肉糜"式发言、歧视性言论。

## 严重程度

| 级别 | 触发条件 |
|---|---|
| 🔵 苗头 | 言论被截图讨论，限于行业小圈子 |
| 🟡 发酵 | 跨圈传播，普通网友开始讨论 |
| 🟠 爆发 | 登上热搜，主流媒体/大V跟进评论 |
| 🔴 危机 | 引发全民声讨、品牌抵制、合作方切割 |

## 应对步骤

### 1. 冷静期
- 不要让当事人继续说话。多说多错。
- 不要让公关人员发"这是个人观点"类切割 — 老板的言论就是公司的言论

### 2. 信息收集
- [ ] 回溯言论的完整上下文（有没有断章取义？）
- [ ] 统计各平台的讨论热度和主流观点
- [ ] 评估对业务的实际影响（合作方反应、销售数据、客户问询）
- [ ] 了解当事人真实意图（说错了？被误解了？还是真的这么想？）

### 3. 定性判断

| 情况 | 策略 |
|---|---|
| 被断章取义/剪辑 | 放完整视频/上下文 + 澄清 |
| 确实说错/口误 | 本人道歉 + 说明本意 |
| 暴露真实价值观问题 | 真诚道歉 + 个人反思 + 行动证明 |
| 陈年旧帖被翻出 | 承认过去的局限 + 现在的改变 |

### 4. 回应策略

**最重要的判断：本人道歉还是官方声明？**

- 口误/被误解 → 本人出镜/发文解释，坦诚直接
- 确实说错了 → 本人道歉，不要代笔感太强
- 价值观翻车 → 本人道歉 + 具体行动，光说没用

**绝对不要：**
- "这是被断章取义"（除非有完整视频铁证）
- "这个言论是XX年前的"（等于承认说过）
- "我道歉如果冒犯了任何人"（条件式道歉 = 没道歉）

### 5. 执行清单
- [ ] 和当事人沟通：先确认他/她的真实态度（愿意道歉还是不服？）
- [ ] 法律评估：有没有违法风险？
- [ ] 撰写道歉/澄清文案
- [ ] 当事人录制视频或发文
- [ ] 后续观察舆论变化
- [ ] 必要时主动接受媒体采访（一个深度访谈比十份声明有用）

## 特别提醒

- 老板言论翻车是最难处理的危机之一，因为品牌=老板人格化
- 语气不对比内容不对更致命。道歉要像真人说话，不要像公关稿
- 一次翻车可以被原谅，反复翻车就是人设崩塌
