# 同行抹黑

## 场景描述
竞争对手通过散布虚假信息、恶意比较、刷差评、挖黑料等手段攻击。典型：竞品搞"XX vs XX"对比黑稿、评论区刷差评、找 KOL 发测评黑稿、挖老板/员工黑料。

## 严重程度

| 级别 | 触发条件 |
|---|---|
| 🔵 苗头 | 出现不正常的竞品对比内容，评论区少量异常 |
| 🟡 发酵 | 多个号同时发类似内容，明显有组织 |
| 🟠 爆发 | KOL/大V 加入，形成舆论话题 |
| 🔴 危机 | 被主流媒体报道，或被平台/监管部门关注 |

## 应对步骤

### 1. 冷静期
不要骂回去。对骂就是给对方送流量。

### 2. 信息收集
- [ ] 收集所有攻击内容的截图（含发布时间、发布者）
- [ ] 分析攻击内容的共同特征（统一话术？同一时间发？同一MCN？）
- [ ] 追查背后推手（发帖号之间的关系、利益关联）
- [ ] 评估攻击内容的杀伤力（哪条最可能让人信？哪条完全站不住脚？）
- [ ] 保存所有证据（用于后续法律手段）

### 3. 定性判断

| 情况 | 策略 |
|---|---|
| 内容完全虚假 | 逐条辟谣 + 展示证据 + 法律警告 |
| 夸大/歪曲事实 | 指出失实部分 + 披露真实情况 |
| 基于真实问题放大 | 先解决真实问题 + 再回应攻击 |
| 对方搞低价/恶意竞争 | 不跟风降价，强调品质和服务差异化 |

### 4. 回应策略

**关键原则：打事实，不打情绪。你用证据说话，对方用情绪骂人，围观者信你。**

- 不要点名对方（"某友商"代替）
- 不要情绪化反击
- 逐条回应最核心的 2-3 条指控，不要面面俱到
- 每条回应附带可验证的证据（检测报告、时间戳、完整对话记录）

### 5. 执行清单
- [ ] 整理证据包（截图、数据、时间线）
- [ ] 写好澄清声明（用 clarification/denial 模板）
- [ ] 评估是否需要发律师函（虚假且造成实际损失才发，否则浪费钱）
- [ ] 通过自有渠道发布（官方号、官网、老板号）
- [ ] 有铁证时可以反向曝光幕后推手
- [ ] 向平台举报有组织的恶意内容

## 特别提醒

- "同行抹黑"的前提是你确认是同行干的。不确定的话先走 viral-misinfo 方案
- 公开爆对方名字是把双刃剑，只有铁证如山才用
- 最好的反击不是吵架，是让你的客户帮你说话
- 律师函慎发。没有后续行动的律师函 = 吓唬人，反而减分
