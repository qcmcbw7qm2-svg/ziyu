# 产品质量问题

## 场景描述
产品出现质量问题，导致消费者投诉、媒体关注或监管介入。如：食品异物、电器安全隐患、产品功能缺陷、材质与宣传不符。

## 严重程度

| 级别 | 触发条件 |
|---|---|
| 🔵 苗头 | 个别用户反馈，未被放大传播 |
| 🟡 发酵 | 社交平台出现集中投诉帖子，被大V/媒体转发 |
| 🟠 爆发 | 主流媒体跟进，登上平台热搜 |
| 🔴 危机 | 监管部门介入调查，或出现人身伤害事件 |

## 应对步骤

### 1. 冷静期
- 🔵🟡：1-2 小时，先内部核实
- 🟠🔴：30 分钟内启动危机小组，1 小时内出第一份声明

### 2. 信息收集
- [ ] 确认出问题的产品批次、型号、数量
- [ ] 追溯供应链/生产环节，定位问题根因
- [ ] 统计受影响消费者人数和损失程度
- [ ] 截图保存所有投诉帖子和媒体报道
- [ ] 联系平台了解删帖/投诉规则

### 3. 定性判断

| 情况 | 策略 |
|---|---|
| 确实是产品质量问题 | 承认 + 召回/退款 + 整改公告 |
| 产品合格但用户使用不当 | 说明正确使用方法 + 强调产品安全标准 |
| 恶意抹黑/竞品伪造 | 出示质检报告 + 追究法律责任 |

### 4. 回应策略

**核心原则：产品出事，先解决问题再解决舆论。**

- 不回避、不甩锅、不说是"个别现象"
- 第一份声明必须包含：承认问题 + 正在采取的措施 + 受影响用户的处理方案
- 后续持续更新整改进展

### 5. 执行清单
- [ ] 召回/退款/赔偿方案确定
- [ ] 写好第一份声明（用 clarification 或 apology 模板）
- [ ] 准备 FAQ 回应常见追问
- [ ] 指定客服对接通道（电话/微信/平台客服）
- [ ] 持续监测舆情变化，24h/48h 出后续公告
- [ ] 整改完成后主动公布结果

## 回应节奏

```
事发当天：第一份声明（承认 + 措施）
第 2-3 天：更新进展（已联系多少用户、已处理多少）
第 7 天：整改方案公布
第 30 天：整改结果 + 第三方检测报告
```
