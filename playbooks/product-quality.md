# 产品质量问题

## 场景描述

产品出现质量问题，导致消费者投诉、媒体关注或监管介入。这是最常见的企业危机类型，也是处理不当概率最高的类型。

常见触发场景：食品异物/变质、电子产品安全隐患/功能缺陷、服装材质与宣传不符、汽车/家电召回、化妆品过敏/超标、保健品功效争议。

**B2B 和 B2C 的区别**：B2C 产品质量事件直接面对消费者情绪，传播快、情绪强。B2B 面对的是客户关系+供应链信任，需要额外考虑客户保密和合同条款。

## 严重程度

| 级别 | 触发条件 | 典型表现 |
|------|----------|----------|
| 🔵 苗头 | 个别用户反馈，未被放大传播 | 客服收到零星投诉、差评 |
| 🟡 发酵 | 社交平台出现集中投诉帖子，被大V/媒体转发 | 小红书/抖音出现多篇避雷帖 |
| 🟠 爆发 | 主流媒体跟进，登上平台热搜 | 媒体报道+评论区冲爆 |
| 🔴 危机 | 监管部门介入调查，或出现人身伤害/死亡事件 | 工商/药监/市监局立案 |

**升级加速器**：
- 涉及母婴/儿童产品 → 严重程度自动升一级
- 视频证据清晰 → 传播速度快 3-5 倍
- 多人同时投诉 → 从苗头到发酵不经过中间阶段

## 应对步骤

### 1. 冷静期
- 🔵🟡：1-2 小时，先内部核实。不要急着公开回应——核实错了再改比不回应更可怕
- 🟠🔴：30 分钟内启动危机小组，1 小时内出第一份声明

冷静期只做一件事：确认产品到底有没有问题。不是确认「舆论怎么说」，是确认「事实是什么」。

### 2. 信息收集
- [ ] 确认出问题的产品批次、型号、数量、生产日期
- [ ] 追溯供应链/生产环节，定位问题根因（原材料？生产工艺？质检漏检？仓储运输？）
- [ ] 统计受影响消费者人数和损失程度（轻伤/重伤/财产损失）
- [ ] 截图保存所有投诉帖子和媒体报道（含发布时间、平台、发布者）
- [ ] 内部确认：这批产品还有多少在市面/在仓库/在运输中
- [ ] 如果是食品/药品/化妆品，立即确认相关国家标准和自身是否合规
- [ ] 联系平台了解删帖/投诉规则（但不要求删帖）

### 3. 定性判断

| 情况 | 策略 | 声明口径 |
|------|------|----------|
| 确实是产品质量问题 | 承认 + 召回/退款 + 整改公告 | 「我们做错了，我们改」 |
| 产品合格但用户使用不当 | 说明正确使用方法 + 强调产品安全标准 | 「您可能用错了，我们帮您」 |
| 恶意抹黑/竞品伪造 | 出示质检报告 + 追究法律责任 | 「这是假的，这是证据」 |
| 个别批次问题，非全量 | 承认该批次问题 + 说明范围 + 不回避 | 「X月X日-X月X日生产的X批次有问题，其他的没问题」 |

**最关键的判断**：坦率的讲，大多数产品质量事件，企业自己心里清楚是不是真的有问题。有问题装没问题，被发现是早晚的事，而且被发现之后的代价是承认的十倍。

### 4. 回应策略

**核心原则：产品出事，先解决问题再解决舆论。产品修不好，公关说破天都没用。**

**第一份声明必须包含三个要素**：
1. 承认问题的存在（不说「个别现象」「用户操作不当」「概率极低」）
2. 正在采取的措施（召回/下架/退款/检测）
3. 受影响用户的处理方案（怎么联系、怎么退、怎么赔）

**声明的语气分级**：

| 严重程度 | 语气 | 负责人出面 |
|----------|------|-----------|
| 🔵 苗头 | 客服口径，一对一处理 | 不需要 |
| 🟡 发酵 | 官方声明，承认问题 | 部门负责人 |
| 🟠 爆发 | 创始人/CEO 出镜道歉或声明 | CEO |
| 🔴 危机 | CEO 出镜+书面声明+政府沟通 | CEO+法务 |

**后续持续更新**：第一份声明之后，公众要的不是你再说一次对不起，是想知道你改得怎么样了。第 3 天、第 7 天、第 30 天各出一份进展通报。

### 5. 执行清单

**即刻**
- [ ] 确认问题批次并立即停产/停售
- [ ] 通知销售渠道下架问题产品
- [ ] 召回/退款/赔偿方案确定（含金额、流程、联系方式）

**声明相关**
- [ ] 写好第一份声明（用 clarification 或 apology 模板，按严重程度选）
- [ ] 准备 FAQ 回应常见追问（「什么时候买的可以退」「怎么退」「有问题的批次怎么查」）
- [ ] 指定客服对接通道（电话/微信/平台客服），确保能打通、有人接

**持续**
- [ ] 24h/48h 出后续进展公告
- [ ] 整改完成后主动公布结果 + 第三方检测报告
- [ ] 持续监测舆情变化
- [ ] 受影响用户一对一跟进完毕

## 回应节奏

```
事发当天：第一份声明（承认 + 措施+ 召回方案）
第 2-3 天：更新进展（已联系多少用户、召回进度、检测进展）
第 7 天：整改方案公布（根因 + 改了什么 + 怎么防止再犯）
第 30 天：整改结果 + 第三方检测报告
```

## 特别提醒

- 产品质量是企业的第一生命线。产品出事，不要试图用公关手段掩盖产品问题。公关只能解决「怎么说」，解决不了「产品烂」
- 「个别现象」是 2026 年最危险的公关用词。大数据时代，是不是个别现象网友自己能扒
- 召回不是失败。召回是负责任的体现。强生召回 3100 万瓶泰诺损失过亿，但市场份额一年内恢复 95%
- 涉及人身伤害时，法律赔偿之外的情绪安抚比法律条款更重要。先关心人，再谈钱
- 母婴/儿童产品的质量事件，不适用以上任何时间表——所有动作必须在 6 小时内完成
