# 创作者被大V引导网暴

## 场景描述

小博主/素人被大V或KOL通过剪辑、断章取义、偷换概念等方式引导粉丝攻击。典型：你的内容被大V截取片段挂出来、观点被曲解后号召举报、被贴上标签（"XX博主"）后跨平台涌入攻击。

**和 online-mob 的区别**：online-mob 是大量陌生人自发的、无组织攻击。这里的攻击是有指挥官的——一个人决定攻击你，N 万人执行。战场不在你这边，在那一个人的评论区里。

**和 viral-misinfo 的区别**：viral-misinfo 是信息被扭曲后自发传播。这里是有意引导。

## 严重程度

| 级别 | 触发条件 |
|---|---|
| 🔵 苗头 | 大V在某条内容中提到你，评论开始出现零星攻击 |
| 🟡 发酵 | 攻击言论明显增加，开始跨平台涌入 |
| 🟠 爆发 | 多个大V/营销号跟进，形成「锤人」话题 |
| 🔴 危机 | 人肉信息泄露、家人被波及、现实威胁出现、或平台开始限流/封禁你的账号 |

## 应对步骤

### 1. 冷静期
**不要在大V评论区吵。在她的主场你永远是输的。**

- 🔵🟡：2-4 小时，收集证据
- 🟠🔴：立即开启证据保全 + 关闭通知，公开回应等 6-12 小时

这期间只做一件事：**证据保全。**

### 2. 信息收集

- [ ] 完整录屏大V的原内容（从开头到结尾，证明她说了什么）
- [ ] 完整保留你自己的原内容（证明你说了什么）
- [ ] 截图大V评论区的高赞攻击言论
- [ ] 保留时间线：你的内容什么时候发的 → 大V什么时候发的 → 攻击什么时候开始的
- [ ] 如果她剪辑了你的内容 → 标注具体剪掉了什么、上下文是什么

### 3. 定性判断

| 情况 | 策略 |
|---|---|
| 对方断章取义/恶意剪辑 | 放完整上下文 + 标注被剪掉的部分 |
| 对方曲解你的观点 | 用一句话澄清核心观点 + 不放逐条辩论 |
| 对方贴标签（"XX博主"） | 用过往内容证明你的真实立场 |
| 你确实说了不当内容 | 认错 + 道歉 + 不说「但是」 |

### 4. 回应策略

**核心原则：把战场从她的评论区拉回你的内容本身。**

**怎么做：**
- 发一条回应，在你的主平台发。不在对方平台发
- 直接放完整上下文（原视频/原文/完整截图）
- 标注被截取/被曲解的具体部分：「她截掉的前半句是XXX，后半句是XXX」
- 不要说「某博主故意害我」——说「这条内容被剪辑后，我的意思被完全反过来了」
- 不要做 10 分钟的逐条反驳。90 秒放完证据就够了

**为什么这个结构有效：**
- 大V靠剪辑制造矛盾 → 你放完整版 → 矛盾消失
- 大V靠情绪带动粉丝 → 你只放事实不动情绪 → 围观者觉得你更可信
- 你不攻击对方 → 对方粉丝找不到「守护我方博主」的战斗理由
- 你提到具体被歪曲的细节 → 围观者自己就能判断

### 5. 执行清单

**证据端**
- [ ] 完整的证据包（双方内容 + 时间线 + 评论截图）
- [ ] 被剪辑的内容 → 标注剪掉了什么
- [ ] 你的过往内容 → 用来证明你的真实立场

**回应端**
- [ ] 发一条回应视频/帖子（90 秒内，完整证据链）
- [ ] 不提对方名字（"某博主"代替）
- [ ] 评论区暂不开或仅关注可评论

**平台端**
- [ ] 向平台举报大V原内容「侵犯名誉权/恶意剪辑/造谣」
- [ ] 如果对方号召举报你 → 向平台说明情况、提交证据

**法律端**
- [ ] 律师评估是否达到名誉权侵权标准（恶意剪辑 + 实际损害 = 可起诉）
- [ ] 如果涉及人肉/人身威胁 → 报警

### 6. 特别提醒

- 在大V的评论区吵架 = 在对方主场打客场，永远输
- 你只有一条回应的机会。第一条最有力，第二条大家就不看了
- 不对等战争的核心逻辑：对方有流量，你有真相。她把真相歪曲了，你放回去。围观者能判断
- 不要指望对方道歉。目的不是让对方认错，是让围观路人看清事实

## 恢复节奏

```
今日：发回应内容 + 关评论
第 2-3 天：发一条正常的、高质量的内容
           不要在内容里提这个事件
第 7 天：评估是否恢复正常
           如果对方追加攻击 → 法律途径
           如果沉默 → 事件结束
```

网暴热度的自然周期是 48-72 小时。只要你不再提供新的素材（第二条回应、哭诉视频、朋友下场等），攻击热度会自行消退。
