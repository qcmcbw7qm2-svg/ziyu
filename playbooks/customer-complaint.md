# 客户投诉发酵

## 场景描述
单个或少量客户的投诉被社交媒体放大，形成集中声讨。典型：客户服务态度差、退款纠纷、承诺未兑现、体验极差。

与产品质量问题的区别：产品质量是「东西坏了」，投诉发酵是「我被坑了」，后者情绪驱动更强。

## 严重程度

| 级别 | 触发条件 |
|---|---|
| 🔵 苗头 | 不满客户发帖/发视频，播放量 < 1 万 |
| 🟡 发酵 | 被同类型消费者共鸣转发，播放量 10 万+ |
| 🟠 爆发 | 出现"XX避雷""XX踩坑"类爆料合集，跨平台扩散 |
| 🔴 危机 | 形成标签/话题，媒体介入做消费者维权报道 |

## 应对步骤

### 1. 冷静期
🔵🟡：先查清楚事情经过，不要一看到差评就冲上去回复。
🟠🔴：1 小时内回应（不需要解决完，但要让外界知道你知道了）。

### 2. 信息收集
- [ ] 还原事情完整经过（客服聊天记录、通话录音、订单记录）
- [ ] 确认己方有没有问题、问题出在哪
- [ ] 追踪传播链条：谁发的？被谁转了？现在哪些平台在讨论？
- [ ] 了解投诉方诉求：要退款？要道歉？要赔偿？

### 3. 定性判断

| 情况 | 策略 |
|---|---|
| 己方确实有错 | 道歉 + 补偿 + 流程整改 |
| 双方各有责任 | 承认己方责任 + 说明对方情况（但不指责） |
| 客户无理取闹 | 说明事实 + 保持克制 + 不激化 |
| 恶意碰瓷/职业差评 | 收集证据 + 平台举报 + 必要时法律手段 |

### 4. 回应策略

**关键判断：公开回应 vs 私下解决。**

- 投诉方愿意沟通 → 优先私下解决，解决后征得同意发和解说明
- 投诉方不沟通/继续升级 → 公开回应，说明事实和解决方案
- 注意：公开回应不等于公开撕逼，永远不对骂

### 5. 执行清单
- [ ] 联系投诉方，了解真实诉求
- [ ] 内部复盘：哪个环节出问题？客服流程？售后政策？
- [ ] 拟定解决方案（道歉+补偿+整改）
- [ ] 写好回应文案（道歉或澄清模板）
- [ ] 解决后请对方确认或发和解声明
- [ ] 内部流程整改，避免同类问题

## 特别提醒

- 不要在评论区跟围观者吵架，一条都不要回
- 不要威胁投诉方（"你等着收律师函"）— 这是火上浇油
- 不要公开说"这个人就是来碰瓷的" — 除非有铁证，否则你就是恶人
- 补偿要到位但不过度（过度补偿会引来更多"碰瓷"）
