# 餐饮食品安全

## 场景描述

餐饮门店出现食品安全或卫生事件被公开曝光，包括但不限于：异物进入食品、宠物进入就餐区接触食材、后厨卫生被拍、食材来源被质疑、顾客用餐后身体不适、监管抽检不合格。

**连锁加盟品牌特别注意**：一家店出事，全网加盟商跟着受损。必须同时对顾客、加盟商、公众三方负责。

## 严重程度

| 级别 | 触发条件 |
|---|---|
| 🔵 苗头 | 个别顾客投诉/差评，无视频传播 |
| 🟡 发酵 | 视频/图片开始在本地传播，被本地生活号转发 |
| 🟠 爆发 | 抖音/小红书爆了，被全国性媒体/大V跟进，加盟商开始收到询问 |
| 🔴 危机 | 登上热搜，监管部门介入调查，或出现顾客群体性身体不适 |

## 应对步骤

### 1. 冷静期
食品安全事件没有真正的冷静期。🟠以上必须 24 小时内回应。但回应前做两件事：
- 确认视频/图片的真实性（不是 P 的、不是旧视频、事发门店确认）
- 不要先回应再说——说错了再改比不回应更可怕

### 2. 信息收集
- [ ] 找到原始视频，确认事发门店、时间
- [ ] 联系门店店长核实情况
- [ ] 调取门店监控录像
- [ ] 联系当事顾客了解诉求
- [ ] 评估是否涉及监管部门（市场监管、卫健委）

### 3. 定性判断

| 情况 | 策略 |
|---|---|
| 门店管理失职（员工未执行卫生标准） | 认错 + 门店整改 + 员工培训 + 规则升级 |
| 顾客行为引发（带宠物、自带食物等） | 承认管理疏漏（未劝阻）+ 规则升级（禁止）+ 不把锅甩给顾客 |
| 食材供应链问题 | 溯源追查 + 下架召回 + 公示供应链 |
| 恶意剪辑/造谣 | 放完整视频 + 法律手段 |
| 顾客身体不适（疑似食物中毒） | 立即配合监管调查 + 承担医疗费用 + 停业配合 |

### 4. 回应策略

**核心原则：食品安全事件不争辩、不解释、不拖延。认错快、行动快、改得快。**

**声明结构（400 字以内）：**
1. 确认事实 — 「经核实，事情发生在X月X日X店」
2. 认错 — 「是我们的管理失职」「我们做错了」
3. 已做的行动 — 联系顾客/停业/消杀/更换/员工培训
4. 规则升级 — 全国门店统一执行新标准
5. 创始人口吻收尾 — 「改不好不开」「你用脚投票是应该的」

**不要在声明里做的事：**
- ❌ 「正在调查中」— 如果视频是真的，不要拖
- ❌ 「个别门店行为」— 加盟商是你授权的
- ❌ 解释为什么发生 — 不重要，公众只关心你怎么改
- ❌ 卖惨 — 「创业不容易」「我们也是受害者」

### 5. 执行清单

**线下**
- [ ] 当事门店停业整改
- [ ] 涉事区域深度消杀 + 餐具更换
- [ ] 当事员工停岗培训（不要开除——开除=甩锅给员工）
- [ ] 全国门店自检自查

**线上**
- [ ] 发一条抖音声明（哪里出事哪里回应）
- [ ] 不关评论区，选择性回复
- [ ] 不联系原发者删视频

**加盟商管理**
- [ ] 当天发内部通告：事件经过 + 处理方案 + 新规则
- [ ] 48h 内召开加盟商线上会议
- [ ] 统一对外口径：「总部已在处理」

**后续节奏**
```
第 2-3 天：发整改过程视频（消杀/培训/设备更换）
第 7 天：整改完毕，检查合格后恢复营业
第 30 天：全国门店统一执行新规则（标识/流程/培训）
```

## 特别提醒

- 食品安全是餐饮的第一红线。产品不好吃可以改，卫生问题触犯的是消费者底线
- 连锁品牌不要甩锅给加盟商。「加盟商个人行为」这句话不发
- 承认管理失职比推卸责任损失小——你认错，消费者反而信你
- 规则升级是最强的安抚：「这种事不会再发生」比「我们很抱歉」有用十倍
