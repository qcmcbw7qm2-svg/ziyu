# 舆情监测操作指南

## 监测对象

根据企业/个人实际情况配置：
- 短视频平台：相关视频评论区 + 话题讨论
- 微博/小红书：搜索品牌名/人名/关键词
- 搜索引擎：搜索主体名称，关注媒体跟进
- 垂类社区：行业论坛、知乎等

## 监测频率

| 阶段 | 频率 | 原因 |
|---|---|---|
| 爆发期（前 72h） | 每 6 小时一次 | 风向变化以小时计 |
| 处理期（3-7 天） | 每天一次 | 关注是否有二次传播 |
| 消退期（一周后） | 每周一次 | 确认彻底平息 |

## 每次监测看什么

### 1. 数据变化
- 原发内容：点赞/转发/评论数变化趋势
- 是否出现新的衍生内容（媒体二次报道、自媒体二创）

### 2. 评论区风向

按以下分类统计热评：

| 类型 | 标志 | 应对 |
|---|---|---|
| 🟢 转移型 | 讨论转移到其他话题/企业 | 安全，不干预 |
| 🟡 质疑型 | "还有其他问题吗""以前有没有类似情况" | 准备背景材料 |
| 🟠 攻击型 | 负面标签化、抵制呼声 | 不回复，观察趋势 |
| 🔴 升级型 | 举报/投诉/报警等具体行动号召 | 确认是否已实际发生 |
| 🔵 科普型 | 评论区自发科普/辟谣 | 良性，不干预 |

### 3. 关键阈值

以下情况触发升级响应：
- 攻击型评论占比超过 30% → 准备二次回应口径
- 衍生内容（二创/媒体报道）超过 3 条 → 评估是否需要主动沟通
- 监管部门公开表态 → 立即配合，主动发布进展公告

## 触发升级的行动清单

- [ ] 启动对应类型 playbook
- [ ] 组织危机应对小组
- [ ] 草拟第一份声明
- [ ] 建立对外统一口径
- [ ] 指定对外沟通渠道
- [ ] 持续监测并记录日志
