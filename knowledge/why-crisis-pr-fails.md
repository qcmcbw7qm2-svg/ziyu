# 危机公关为什么失败 — 9 种经典败局

来源：田辉《危机公关：逆境中如何掌握声誉主导权》第二章案例复盘

## 1. 公关失能 — 逃避责任 + 沟通失灵

**案例**：西安奔驰车主维权（2019）
**表现**：拖延躲避、言行不一、对消费者强势、单方面宣布已达成一致被当事人否认
**教训**：承担责任、同感共情、言行一致。面对消费者时，情绪管理比事实分析更优先

## 2. 仓促上阵 — 准备不足

**案例**：碧桂园系列安全事故（2018）
**表现**：全国媒体见面会变"免费旅行"、高管发言翻车、伴手礼合规风险
**教训**：媒体关系功夫在平时，不要临阵磨枪；发言人必须经过专业培训；危机时刻重点是解决问题，不是秀全方位企业形象

## 3. 过度自信 — 口无遮拦

**案例**：英国石油公司墨西哥湾漏油（2010）
**表现**：CEO 公开说"影响不大""希望我的生活早日回归正轨"、国会问询一问三不知
**教训**：CEO 不等于公关专家。危机中有两条线——危机现场线和舆情现场线，顾此失彼会满盘皆输。高管一言一行都在被直播

## 4. 单打独斗 — 体系缺失

**案例**：携程幼儿园虐童风波（2017）
**表现**：公关总监发帖灭火、HR 要求员工封口、最高管理层 5 天后才介入
**教训**：公关部必须是危机应对协调人。平战结合：平时搭建预警机制，战时挺身而出。部门合力 > 单打独斗

## 5. 价值错位 — 生命至上不可谈判

**案例**：滴滴顺风车两起命案（2018）
**表现**：第一次承诺整改未落实、承诺"3 倍赔偿"被批漠视生命、客服机械回复错过报警窗口
**教训**：涉及人命时，任何公关技巧都失去意义。道歉不能替代整改。商业道德和价值观是公关的底线，破了这个底线，说什么都没用

## 6. 态度变形 — 道歉信本身翻车

**案例**：范冰冰偷逃税（2018）、翟天临学术造假（2019）
**表现**：道歉信出现语法错误/抄袭模板/避重就轻/不承认核心问题
**教训**：道歉信是细节活。语法错误 = 态度不端正。最关键的是直面核心问题，不要绕

## 7. 跨越红线 — 公器私用

**案例**：鸿茅药酒跨省抓捕医生（2018）
**表现**：因一篇阅读量 2241 的文章，动用警方跨省抓捕作者
**教训**：法律是公关的最高红线。"公安"不能干"公关"的活。品牌传播必须遵守真实性原则

## 8. 认知错位 — 对公关的理解就错了

**普遍现象**：
- "公关就是搞关系/发稿子/搞定媒体"
- "公关部就是花钱的部门/办活动的/帮老板写稿子的"

**正确的公关认知**：
- 公关 = 在组织与公众间建立互利关系的战略性沟通过程
- 公关有两个核心职能：管理 + 决策
- 传播是手段，不是目的
- 声誉是结果，不是可以直接追逐的目标

## 9. 利益相关方失明 — 不知道对谁负责

**核心概念**：利益相关方是与企业存在利益关系或影响关系的人/机构/群体
- 主要：消费者、投资人、员工、供应商
- 次要：媒体、公众、竞争对手、政府

**关键判断**：从危机管理角度看，没有主次之分。任何一个利益相关方都可能引发危机或影响危机走向。危机决策的出发点应该是利益相关方，不是企业短期利益

## 总结：为什么失败

| 失败类型 | 根因 | 一句话 |
|---|---|---|
| 公关失能 | 能力缺失 | 不会做 |
| 仓促上阵 | 准备缺失 | 没准备好 |
| 过度自信 | 认知错位 | 太把自己当回事 |
| 单打独斗 | 体系缺失 | 没人配合 |
| 价值错位 | 底线失守 | 价值观错了 |
| 态度变形 | 细节失误 | 道歉都不会 |
| 跨越红线 | 法律踩线 | 公器私用 |
| 认知错位 | 根本不懂公关 | 方向错了 |
| 利益相关方失明 | 不知道对谁负责 | 对象错了 |
