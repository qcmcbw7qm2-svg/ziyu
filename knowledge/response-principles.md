# 危机应对基本原则

## 危机的本质

- 危机不可避免（如同纳税和死亡）
- 危机无法预测走向（传播的不确定性 × 应对的不确定性）
- 危机影响巨大（经济利益 + 企业声誉 + 员工士气 + 长期发展）
- 危中有机（危险与机遇并存，可以转化）

## 危机的生命周期（9 阶段）

```
酝酿期 → 显现期 → 发生期 → 爬坡期 → 爆发期 → 处理期 → 衰退期 → 静默期 → 修复期
```

关键节点：
- **爬坡期**是止损的最佳窗口。错过这个窗口，修复成本指数级上升
- **爆发期**的标志：主流媒体报道、登上热搜、政府部门介入
- **静默期**是危机处理中最容易被忽视但最关键的环节。不要急于发声洗白
- **修复期**：品牌修复需要数月甚至数年，不可急于求成

## 危机的两条战线

| 战线 | 任务 | 失败后果 |
|---|---|---|
| 危机现场线 | 控制事态发展，降低实际损害 | 损失扩大 |
| 舆情现场线 | 通过直达人心的回应进行情绪管理 | 舆论失控 |

两条线必须同时打赢。只顾现场不管舆情 = 做了事没人知道。只顾舆情不管现场 = 空心公关。

## 核心原则

### 1. 速度 > 完美
舆论发酵以小时计。24 小时内不出声，舆论场就会被对手/围观者填满。第一份回应不必完美，但必须快。

### 2. 事实 > 态度
大众不需要你的"态度诚恳"，需要的是：发生了什么？影响多大？你打算怎么办？先给事实，再谈态度。

### 3. 行动 > 语言
"我们正在调查"比"我们深感抱歉"有用十倍。回应里必须有具体行动、时间节点、责任人。

### 4. 一个声音
危机期间，只有一个人/一个渠道对外发声。多人多话 = 口径不一 = 次生危机。

## 危机分级

| 级别 | 定义 | 示例 | 响应速度 |
|---|---|---|---|
| 🔵 苗头 | 个别用户抱怨，未被扩散 | 评论区差评 | 24h 内回复 |
| 🟡 发酵 | 被截图转发，KOL 开始关注 | 小红书笔记爆了 | 6h 内回应 |
| 🟠 爆发 | 多平台扩散，媒体介入 | 上热搜/上新闻 | 2h 内回应 |
| 🔴 危机 | 监管部门介入/大规模抵制 | 工商介入/全网声讨 | 立即 |

## 5 步应对法

```
冷静期 → 信息收集 → 定性判断 → 策略选择 → 执行清单
```

### 冷静期
情绪最激动的时候不做任何公开回应。发出去的删不掉。

### 信息收集
- 舆论从哪开始？谁引爆的？
- 核心指控是什么？（剥离情绪，提炼事实）
- 传播到哪些平台了？量级多大？
- 有没有法律风险？

### 定性判断
这张表决定后续所有动作：

| 指控性质 | 应对方向 |
|---|---|
| 事实属实 | 承认 + 补救 + 整改 |
| 部分属实 | 承认对的 + 澄清错的 + 补救 |
| 完全失实 | 澄清事实 + 出示证据 |
| 恶意造谣 | 辟谣 + 法律手段 |

### 策略选择
- **正面回应**：事实层面的指控，有证据可出示
- **冷处理**：纯粹情绪发泄，无事实指控，回应反而放大
- **转移焦点**：发布正面信息稀释（慎用，容易被骂"转移视线"）
- **法律手段**：造谣诽谤、恶意竞争，律师函/报警

### 执行清单
- [ ] 指定唯一发言人
- [ ] 内部统一口径
- [ ] 起草回应文案
- [ ] 法律审核（有法律风险时）
- [ ] 选定发布渠道和时间
- [ ] 准备二次回应预案（回应后的常见追问）
- [ ] 关闭评论区或准备评论管理策略
- [ ] 监测回应后的舆论变化

## 危机决策的出发点：利益相关方

危机时刻做决策，问自己：受这件事影响的人/群体是谁？

| 利益相关方 | 与企业的关系 | 危机中他们关心什么 |
|---|---|---|
| 消费者/客户 | 直接经济利益 | 我的权益有没有保障？ |
| 员工 | 直接经济利益 + 情感 | 公司会怎么处理？我会不会受影响？ |
| 投资人/股东 | 直接经济利益 | 损失多大？股价会跌多少？ |
| 供应商/经销商 | 直接经济利益 | 会不会牵连到我？ |
| 媒体 | 传播影响力 | 有什么可以报道的？ |
| 政府/监管 | 执法与政策 | 有没有违法？要不要介入？ |
| 公众 | 认知与评判 | 这家公司值不值得信任？ |

危机决策的优先级 = 利益相关方利益 > 企业短期利益。强生泰诺案的核心：先保护公众，再保护企业，最终两者都保住了。

## 强生泰诺案 — 危机公关的黄金标准

1982 年，7 人因服用被投毒的强生泰诺胶囊死亡。强生的做法：

1. **全美召回 3100 万瓶**（损失超 1 亿美元）— 承担责任的极致
2. **CEO 上电视**（之前 30 年从未接受采访）— 开放姿态
3. **邀请 CBS《60分钟》拍摄内部会议**— 透明到底
4. **推出三重安全包装**— 用行动解决根本问题
5. **卫星直播全国新闻发布会**— 权威证实

结果：一年内市场份额恢复到 95%。从此成为所有危机公关教材的第一课。

**5S 原则**（从泰诺案提炼）：
- **Speed**（速度）：快速回应
- **Sincerity**（真诚）：开诚布公
- **Shoulder the matter**（承担责任）：不推诿
- **System**（系统处理）：团队作战，不是单打独斗
- **Standard**（权威证实）：借第三方声音增加可信度

但 5S 之上，最核心的是 **Openness（开放的姿态）**。这是走对危机公关第一步的根本。

## 常见错误

- ❌ 第一时间说"这是误会" — 没搞清楚状况就否认，万一后面反转就是二次翻车
- ❌ 长文解释 — 大众只看标题和前两段，长文=心虚
- ❌ 删评论 — 删评=坐实，截图会比删得快
- ❌ 开小号洗地 — 一被发现就是"水军实锤"
- ❌ 攻击提出质疑的人 — 把事实之争变成情绪对立

## 4A 道歉法

来源：田辉《危机公关》第三章。一份有效的危机回应需包含四个层次：

| 步骤 | 英文 | 含义 | 关键词 |
|------|------|------|--------|
| 1. 应答 | Acknowledge | 告知公众已注意到此事 | "我们注意到了""我们已展开调查" |
| 2. 道歉 | Apology | 对造成的伤害表示歉意 | "情况属实""我们真诚道歉" |
| 3. 态度 | Attitude | 表达企业的立场和认知 | "出现错误的原因在于……" |
| 4. 行动 | Actions | 具体的整改措施和时间表 | "我们将采取以下行动……" |

**四个层次缺一不可。** 只说"我们注意到了"不给行动 = 敷衍。直接跳到行动不说态度 = 冰冷。有态度没行动 = 空话。

**海底捞的标杆案例**：2017年卫生事件，两封致歉信完美执行了 4A——第一封认错道歉（应答+道歉+态度），第二封列出 7 条整改措施+责任人+电话（行动）。被网友总结为"这锅我背、这错我改、员工我养"。

**反面教材**：碧桂园 2018 年"7·26"事故声明，只描述了事故过程（特大暴雨和瞬时大风），缺少对受害者家属的情绪关怀。被批"声明冰冷"。

## 危机等级与响应路径

| 等级 | 定义 | 处理人 | 响应时间 |
|------|------|--------|----------|
| 小事件 | 个体投诉/纠纷，未扩散 | 一线经理/客服 | 即时处理 |
| 潜在危机 | 可能引发关注的事件 | 部门负责人+公关部 | 2-4 小时内 |
| 危机 | 已被媒体关注报道 | 危机应对小组 | 24 小时内回应 |
| 大型危机 | 全网红/监管介入 | 最高管理层+危机小组 | 立即响应 |

**关键洞察**：从小事件到大型危机，往往只差一个处理不当。英国石油公司墨西哥湾漏油，起因只是海底油井水泥不符合规范。

## 危机管理团队构成

核心成员（必须）：
- 最高决策者（拍板人）
- 公关负责人（召集人+协调者）
- 法务负责人
- 相关职能部门负责人

扩展成员（按需）：
- 政府关系负责人
- 人力资源负责人
- 相关事业部/区域负责人

**核心原则**：公关部是危机应对的协调人，不是唯一责任人。危机是企业的危机，公关部只是牵头组织。

## 模拟演练

企业应每两年进行 1-2 次危机公关模拟演练，包括：
- 场景设定 + 角色扮演
- 回应声明撰写
- 模拟媒体采访（发言人培训）

演练后必须复盘，更新危机管理手册。
