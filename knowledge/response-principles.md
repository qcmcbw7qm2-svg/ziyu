# 危机应对基本原则

## 危机的本质

- 危机不可避免（如同纳税和死亡）
- 危机无法预测走向（传播的不确定性 × 应对的不确定性）
- 危机影响巨大（经济利益 + 企业声誉 + 员工士气 + 长期发展）
- 危中有机（危险与机遇并存，可以转化）

## 危机的生命周期（9 阶段）

```
酝酿期 → 显现期 → 发生期 → 爬坡期 → 爆发期 → 处理期 → 衰退期 → 静默期 → 修复期
```

关键节点：
- **爬坡期**是止损的最佳窗口。错过这个窗口，修复成本指数级上升
- **爆发期**的标志：主流媒体报道、登上热搜、政府部门介入
- **静默期**是危机处理中最容易被忽视但最关键的环节。不要急于发声洗白
- **修复期**：品牌修复需要数月甚至数年，不可急于求成

## 危机的两条战线

| 战线 | 任务 | 失败后果 |
|---|---|---|
| 危机现场线 | 控制事态发展，降低实际损害 | 损失扩大 |
| 舆情现场线 | 通过直达人心的回应进行情绪管理 | 舆论失控 |

两条线必须同时打赢。只顾现场不管舆情 = 做了事没人知道。只顾舆情不管现场 = 空心公关。

## 核心原则

### 1. 速度 > 完美
舆论发酵以小时计。24 小时内不出声，舆论场就会被对手/围观者填满。第一份回应不必完美，但必须快。

### 2. 事实 > 态度
大众不需要你的"态度诚恳"，需要的是：发生了什么？影响多大？你打算怎么办？先给事实，再谈态度。

### 3. 行动 > 语言
"我们正在调查"比"我们深感抱歉"有用十倍。回应里必须有具体行动、时间节点、责任人。

### 4. 一个声音
危机期间，只有一个人/一个渠道对外发声。多人多话 = 口径不一 = 次生危机。

## 危机分级

| 级别 | 定义 | 示例 | 响应速度 |
|---|---|---|---|
| 🔵 苗头 | 个别用户抱怨，未被扩散 | 评论区差评 | 24h 内回复 |
| 🟡 发酵 | 被截图转发，KOL 开始关注 | 小红书笔记爆了 | 6h 内回应 |
| 🟠 爆发 | 多平台扩散，媒体介入 | 上热搜/上新闻 | 2h 内回应 |
| 🔴 危机 | 监管部门介入/大规模抵制 | 工商介入/全网声讨 | 立即 |

## 5 步应对法

```
冷静期 → 信息收集 → 定性判断 → 策略选择 → 执行清单
```

### 冷静期
情绪最激动的时候不做任何公开回应。发出去的删不掉。

### 信息收集
- 舆论从哪开始？谁引爆的？
- 核心指控是什么？（剥离情绪，提炼事实）
- 传播到哪些平台了？量级多大？
- 有没有法律风险？

### 定性判断
这张表决定后续所有动作：

| 指控性质 | 应对方向 |
|---|---|
| 事实属实 | 承认 + 补救 + 整改 |
| 部分属实 | 承认对的 + 澄清错的 + 补救 |
| 完全失实 | 澄清事实 + 出示证据 |
| 恶意造谣 | 辟谣 + 法律手段 |

### 策略选择
- **正面回应**：事实层面的指控，有证据可出示
- **冷处理**：纯粹情绪发泄，无事实指控，回应反而放大
- **转移焦点**：发布正面信息稀释（慎用，容易被骂"转移视线"）
- **法律手段**：造谣诽谤、恶意竞争，律师函/报警

### 执行清单
- [ ] 指定唯一发言人
- [ ] 内部统一口径
- [ ] 起草回应文案
- [ ] 法律审核（有法律风险时）
- [ ] 选定发布渠道和时间
- [ ] 准备二次回应预案（回应后的常见追问）
- [ ] 关闭评论区或准备评论管理策略
- [ ] 监测回应后的舆论变化

## 危机决策的出发点：利益相关方

危机时刻做决策，问自己：受这件事影响的人/群体是谁？

| 利益相关方 | 与企业的关系 | 危机中他们关心什么 |
|---|---|---|
| 消费者/客户 | 直接经济利益 | 我的权益有没有保障？ |
| 员工 | 直接经济利益 + 情感 | 公司会怎么处理？我会不会受影响？ |
| 投资人/股东 | 直接经济利益 | 损失多大？股价会跌多少？ |
| 供应商/经销商 | 直接经济利益 | 会不会牵连到我？ |
| 媒体 | 传播影响力 | 有什么可以报道的？ |
| 政府/监管 | 执法与政策 | 有没有违法？要不要介入？ |
| 公众 | 认知与评判 | 这家公司值不值得信任？ |

危机决策的优先级 = 利益相关方利益 > 企业短期利益。强生泰诺案的核心：先保护公众，再保护企业，最终两者都保住了。

## 强生泰诺案 — 危机公关的黄金标准

1982 年，7 人因服用被投毒的强生泰诺胶囊死亡。强生的做法：

1. **全美召回 3100 万瓶**（损失超 1 亿美元）— 承担责任的极致
2. **CEO 上电视**（之前 30 年从未接受采访）— 开放姿态
3. **邀请 CBS《60分钟》拍摄内部会议**— 透明到底
4. **推出三重安全包装**— 用行动解决根本问题
5. **卫星直播全国新闻发布会**— 权威证实

结果：一年内市场份额恢复到 95%。从此成为所有危机公关教材的第一课。

**5S 原则**（从泰诺案提炼）：
- **Speed**（速度）：快速回应
- **Sincerity**（真诚）：开诚布公
- **Shoulder the matter**（承担责任）：不推诿
- **System**（系统处理）：团队作战，不是单打独斗
- **Standard**（权威证实）：借第三方声音增加可信度

但 5S 之上，最核心的是 **Openness（开放的姿态）**。这是走对危机公关第一步的根本。

## 常见错误

- ❌ 第一时间说"这是误会" — 没搞清楚状况就否认，万一后面反转就是二次翻车
- ❌ 长文解释 — 大众只看标题和前两段，长文=心虚
- ❌ 删评论 — 删评=坐实，截图会比删得快
- ❌ 开小号洗地 — 一被发现就是"水军实锤"
- ❌ 攻击提出质疑的人 — 把事实之争变成情绪对立
