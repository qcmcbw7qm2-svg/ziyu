## 八、关键结论

### 做对了什么

1. **24 小时内回应**，不拖不等不调查
2. **认错不甩锅**，没说「是个别加盟商行为」
3. **规则升级** — 「宠物禁入」给了公众防止再犯的承诺
4. **狗主人致歉** — 外部助力替品牌分担火力
5. **不说套话** — 「改不好不开」比「深感抱歉」管用

### 可复用的框架

```
餐饮食品安全事件 回应公式 =
    确认事实（不拖）
    + 认错（不甩锅）
    + 行动（先做再说）
    + 规则升级（不会再有下次）
    + 创始人口吻（不说套话）
```
