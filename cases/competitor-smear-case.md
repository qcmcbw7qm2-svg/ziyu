# 案例：某护肤品牌被竞品对比抹黑

**日期**：2025-12
**类型**：竞品恶意抹黑
**严重程度**：🟡 发酵 → 及时处理未升级
**对应 playbook**：competitor-smear

## 事件

某国货护肤品牌（年销 5 亿）的核心爆品，被一个竞品在抖音和小红书上同时发起「VS 对比」内容。竞品的 KOL 统一用了同一个话术——「XX 品牌的面霜含 XX 成分，长期使用会导致皮肤敏感」。内容形式高度一致，发布时间集中在 48 小时内，疑似 MCN 统一派单。

## 品牌方的应对

**没有做的事**：
- 没有点名回应对方（说「某友商」代替）
- 没有让自家 KOL 发「对比回去」的视频
- 没有在评论区跟对方粉丝吵架

**做了的事**：
- 48 小时内发布了一份成分安全说明，附 SGS 第三方检测报告
- 报告放在官网可公开访问的位置，不压缩包
- 用旗下 KOL 发科普内容讲「XX 成分的实际安全性」，不提竞品
- 在电商详情页增加了成分安全说明板块

**为什么有效**：不吵架，只放证据。围观者看到检测报告自己会判断。竞品的统一话术反而因为高度一致被网友置疑「是不是收了钱」。

## 经验

1. 竞品攻击最怕的不是你骂回去，是你放证据
2. 同一话术高度一致 = 最容易识别的水军特征。截图存证据
3. 不在对方的主场打——对方选了抖音和小红书，你就在所有平台统一回应
4. 不要点名。你说「某友商」，围观者自己去对号入座，效果比你点名更好
