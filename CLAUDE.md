# 公关舆情处理项目

## 定位

按舆情类型分类的应对方案库。每种类型一套方案，遇到什么看什么。

## 目录约定

| 目录 | 放什么 | 命名规则 |
|---|---|---|
| `playbooks/` | 按舆情类型的应对方案 | `type-name.md` |
| `templates/` | 公关文案半成品模板 | `document-type.md` |
| `cases/` | 真实或收集的案例 | `YYYY-MM-DD-short-title.md` |
| `monitoring/` | 舆情监测配置和记录 | 按功能命名 |
| `knowledge/` | 方法论文档和参考 | 按主题命名 |

## 应对方案结构

每个 playbook 固定包含：
1. 场景描述
2. 严重程度判断标准
3. 应对步骤（冷静期 → 信息收集 → 定性 → 策略选择 → 执行清单）

## 文案原则

- 不写套话（"深感抱歉""深刻反思"类禁用）
- 认事实不认情绪
- 不同的危机类型匹配不同的文案策略
- 模板是半成品，留占位符 `【】`，使用时填充

## 新增规则

- 遇到 playbook 没覆盖的舆情类型 → 新建 playbook
- 收集到典型案例 → `cases/YYYY-MM-DD-short-title.md`
- 文案模板不够用 → `templates/` 下新增
